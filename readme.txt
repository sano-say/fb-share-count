A plugin adds a function that shows Facebook share number with v2.7 Facebook API.

== Description ==

This plugin adds a function that shows Facebook share number.

`<?php fb_share_count(get_the_permalink()); ?>`

Also used as shortcode.

`[fb_share_count]`

* Fill "App ID" and "App secret" at admin panel.
* Call function or shortcode then get FB share count.
